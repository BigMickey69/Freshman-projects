About me...


我是成大工科系大一的石維廉，
Discord Clouding 這個 project 是程式歷程中很多個第一次。 第一次接觸伺服器前端 + 後端，第一次使用 HTML、Cookies、CSS，第一次接觸 Javascript 第一次操作 Discord Bot 與 Discord的API， 第一次使用 SQL，還有之前稍微碰過但摸不著頭緒的線程Threading 還有一堆雜七雜八的import函式庫...

目前Discord Ck僅能算是版本v0.1，還只是一個巨大 project 中的雛型， 儘管如此，程式碼已經約1000行。 本project 很多方面是零基礎邊學邊做， 全程沒有使用Co-Piolet，因此一些地方尤其是細節上的處理有待加強， 但即使這樣，目前的系統已經能實現無限大的雲端空間。
(P.S.有仔細看過 Discord 的 EULA，而 "Discord Clouding" 沒有違反使用者條約)


原本是希望這個雛形就能完全同步顯示檔案的上傳與下載時顯示進度條， 但不幸目前的程式碼與SocketIO似乎有甚麼衝突，幾天泡在 Youtube + 爬文最後仍撞牆而遲遲無法解決， ChatGPT、Gemini也無法找到問題，猜測需要從根本修正一些implementation才行。 最後選擇讓頁面每3.5秒刷新一次來更新這資訊，對使用者而言這做法是沒什麼問題，不過真正的同步才是理想， 目前試過Eventlet 與 Monkey patch，而現在正在研究 Celery 的可能性。

我希望Discord Clouding 可以是個長久的project，願景是讓它成為一個 All-in-one 網站， 提供使用者一個免費且沒有廣告的綠洲，也提供多面向多功能服務。
